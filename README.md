# Мій додаток "online_chat"
Цей дотаток дозволяє спілкуватися корестувачам використовуючи клієнт-серверну систему
Для створеня було використано такі бібліотекі як: - **customtkinter**, **socket**, **threading**
## Як запустити та встановити програму
1. Скачати програму, сервурну частину та exe файл,
2. Запустити exe файл,
3. Заповтини рядки.
